# yuichan

Yui Language - A programming language implementation

## Installation

You can install yuichan from PyPI:

```bash
pip install yuichan
```

## Usage

```python
import yuichan

# Check version
print(yuichan.__version__)
```

## Development

To install in development mode:

```bash
pip install -e .
```

To build the package:

```bash
pip install build
python -m build
```
